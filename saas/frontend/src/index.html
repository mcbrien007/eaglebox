EagleBox is a peer-to-peer, end-to-end encrypted file sharing network. Browse what's shared publicly — only the key unlocks the content.

🔐 AES-256-GCM 🌐 Decentralized 🦅 Kazaa-style index 🔑 Key-gated downloads ✅ Integrity verified
🌐

Public index, private content

Filenames and metadata are visible to everyone on the network. The actual file is encrypted — only the share key decrypts it.

🔐

Zero-knowledge server

Encrypted chunks are stored server-side. The decryption key never leaves your browser. Even we can't read your files.

⚡

P2P + HTTP hybrid

Desktop app users share directly peer-to-peer via Hyperswarm. Web users upload through the relay — same encryption, same index.
